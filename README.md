# Department of Homeland Security (department-of-homeland-security)
The U.S. Department of Homeland Security (DHS) is a cabinet-level federal agency responsible for protecting the nation from terrorism, securing borders, enforcing immigration law, responding to disasters, and securing cyberspace. DHS exposes APIs across its operational components, including FEMA's OpenFEMA platform, USCIS's Developer Portal, the CISA Known Exploited Vulnerabilities catalog, the National Terrorism Advisory System (NTAS) feed, and the DHS Open Data Catalog.

## APIs

### OpenFEMA API
Free public REST API exposing 70+ FEMA datasets including disaster declarations and public assistance.

**Human URL:** [OpenFEMA](https://www.fema.gov/about/openfema)

### USCIS Case Status API
Programmatic lookup of USCIS case status by receipt number.

**Human URL:** [USCIS Case Status API](https://developer.uscis.gov/api/case-status)

### USCIS FOIA Request and Status API
Programmatic submission and tracking of FOIA requests.

**Human URL:** [USCIS FOIA API](https://developer.uscis.gov/api/foia-request-and-status)

### CISA Known Exploited Vulnerabilities Catalog Feed
JSON and CSV feed of CVEs known to be actively exploited.

**Human URL:** [CISA KEV](https://www.cisa.gov/known-exploited-vulnerabilities-catalog)

### National Terrorism Advisory System Feed
XML feed of current NTAS alerts and bulletins.

**Human URL:** [NTAS](https://www.dhs.gov/ntas-api-documentation)

### DHS Open Data Catalog
DHS datasets on Data.gov via CKAN.

**Human URL:** [DHS Data](https://www.dhs.gov/data)
